# timer

To install:

###### on MacOS
1. open terminal
2. run ```curl https://www.python.org/ftp/python/3.10.4/python-3.10.4-macos11.pkg```
3. run ```./python-3.10.4-macos11.pkg```
4. after installation is done, run ```python3 -m pip install flask```
5. to start the app run the file named ```app.py```
