# timer

To install:

###### on MacOS
1. open terminal
2. run ```git clone https://github.com/Asdrei/timer.git timer && cd timer```
3. If python is not installed:
    * run ```curl https://www.python.org/ftp/python/3.10.4/python-3.10.4-macos11.pkg```
    * run ```./python-3.10.4-macos11.pkg```
    * restart
3. run ```python3 -m pip install flask```
4. after installation is done, run ```python3 -m pip install flask```
5. to start the app run the file named ```app.py```
