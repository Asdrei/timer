# timer

## Prerequisites:
* having the latest python3 version installed (can be obtained at https://www.python.org/downloads/)
* having installed flask using  ```python3 -m pip install flask```


## Installation
1. open terminal
2. clone repository using ```git clone https://github.com/Asdrei/timer.git timer && cd timer```
4. to run the timer execute ```python3 app.py``` or ```python app.py``` depending on your python installation
5. follow the address to open the timer in the browser (default is 127.0.0.1:5000)
